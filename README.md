## Kotlin

#### Checklist

- [ ] **Basics**
  - [ ] Variables
    - [ ] `var`
    - [ ] integer, float, string, char, boolean
    - [ ] predefined type -> `var someInt : Int = 5`
  - [ ] Constants
    - [ ] `val`
  - [ ] Comments
  - [ ] Array (limited capacity)
  - [ ] List (no capacity limits)
  - [ ] Set (unique items)
  - [ ] Map (key value)
  - [ ] Operators
  - [ ] String Templates
  - [ ] Conditionals
    - [ ] If statement
    - [ ] Switch
  - [ ] Loops
    - [ ] For
    - [ ] While
  - [ ] Functions
  - [ ] Range
    - [ ] 'in' operator
  - [ ] Collections
    - [ ] filter
  - [ ] Package
  - [ ] Class
    - [ ] 'is' operator
  - [ ] Inheritance
